-----------------------LEEME-----------------------
Este proyecto usa el modelo MVC para implementar un CRUD usando JAVA y MySQL.
Mi base de datos esta basada en los primeros pokemon de agua de la primera generacion de Pokemon,
con los campos:
Numero de Dex Nacional
Nombre
Tipo
Peso
Altura
Ademas de un campo Primary Key AUTO_INCREMENT
He dejado el archivo para la creacion de la BD y asi sea mas facil que se implemente el projecto.

-----------------READ ME-----------------
This project use the MVC Model to implement a CRUD example using JAVA and MySQL
The DB is based of the first 26 WATER of the 1st gen of Pokemon
The colums are:
Number in National Dex
Name
Type
Weight
Height
I defined a special column (idPokemon) as PRIMARY_KEY & AUTO_INCREMENT.
I attach the SQL file (bd-pokemon.sql), in case you need to use this proyect.
You will see that I use Servlets, JSP, and others technologies.
Have a good day!
